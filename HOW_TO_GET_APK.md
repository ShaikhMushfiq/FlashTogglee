# How to get your FlashToggle APK — 5 minutes, no coding

## You need: A free GitHub account
Sign up at https://github.com if you don't have one.

---

## Step 1 — Create a new repository
1. Go to https://github.com/new
2. Repository name: `FlashToggle`
3. Keep it **Public**
4. Click **"Create repository"**

---

## Step 2 — Upload the project files
On the next page, click **"uploading an existing file"**

Drag and drop ALL the files and folders from inside the `FlashToggle/` folder you downloaded.
Make sure you include:
- `.github/` folder (this is the one that triggers the build!)
- `app/` folder
- `build.gradle`
- `settings.gradle`
- `gradle.properties`
- `gradlew`
- `gradle/` folder

Click **"Commit changes"**

---

## Step 3 — Watch it build
1. Click the **"Actions"** tab at the top of your repository
2. You'll see "Build APK" running with a yellow spinner ⏳
3. Wait about 3–4 minutes for it to finish ✅

---

## Step 4 — Download your APK
1. Click on the completed "Build APK" run
2. Scroll to the bottom → **Artifacts**
3. Click **"FlashToggle-APK"** to download
4. Unzip it → you'll find `app-debug.apk`

---

## Step 5 — Install on your phone
1. Transfer `app-debug.apk` to your phone (WhatsApp, email, USB, Google Drive — anything)
2. On your phone: Settings → Security → **"Install unknown apps"** → enable for your file manager/browser
3. Tap the APK file → Install

---

## Step 6 — Set up after install
1. Open **FlashToggle** app
2. Tap **"Start Service"**
3. Go to Settings → Battery → Battery Optimization → FlashToggle → **Don't optimize**
4. **Xiaomi/Redmi only**: Settings → Apps → FlashToggle → Autostart → **Enable**

Now lock your screen and **double-press the power button** — flash toggles! 🔦
